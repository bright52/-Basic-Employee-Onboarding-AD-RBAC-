# Basic Employee Onboarding (AD)(RBAC)

## Problem Statement
* The problem in this project concerned a fictional company called Northstar Medical Group. Their hiring process is messy and all over the place. They had an MSP handling their infrastructure but couldn't keep up as the business expanded. New employees are being added manually, departments are not clearly labeled or organized, access assignments are inconsistent, and accounts are over-privileged. If they were audited now, they would be in huge trouble with HIPAA and PHI compliance regulations.


## Solution Overview
* I built a basic employee lifecycle pipeline using Active Directory as the identity structure for Northstar Medical Group. I set up the Role-Based Access Control (RBAC) matrix and ensured that users were granted access ONLY according to their role. I also simulated a ticket in which a user was provisioned with the incorrect level of access.
